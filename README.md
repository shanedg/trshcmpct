> all build config, no substance

This [monorepo] (mono-repository) is managed with [@microsoft/rush]

## Quick Start

Install rush globally

```sh
npm install --location=global @microsoft/rush
```

Install all project dependencies

```sh
rush install
```

Build all projects

```sh
rush build
```

Test all projects

```sh
rush test
```
