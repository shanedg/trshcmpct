This repository is managed by [Rush](https://rushjs.io/pages/developer/new_developer/).

```sh
npm i -g @microsoft/rush
```

Install all project dependencies.

```sh
rush install
```

Build all projects.

```sh
rush build
```

Lint all projects.

```sh
rush lint
```

Run all project tests.

```sh
rush test
```

Type check all TS projects.

```sh
rush type-check
```
